About This Website

This summer, I wanted to build on my programming skills by building an application from start to finish. This website, which is the result of my learning, illustrates details about the activities that I enjoy. It was built using Node.js, Javascript, CSS3, and HTML5, and is hosted on Heroku. Node.js, a Javascript backend framework, was used in conjunction with Javascript to route URLs to the appropriate HTML pages; CSS was used to enhance the presentation of the HTML pages.
